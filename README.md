## Features

Basic commands include:
* copy
* cut
* delete
* find
* format
* go to line
* hello world
* paste
* quick open
* redo
* search
* select all
* stop listen
* undo

## Extension Settings
None

## Requirements
Other than Windows, this extension needs JRE (JDK for MacOS) to run

## Known Issues
Context menu could hold the commands' actions
